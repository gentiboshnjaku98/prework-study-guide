HTML

  • HTML is the fundamental language for creating and structuring web pages.
  • HTML is a markup language, which means it uses tags to define elements and structure content..

CSS

  • CSS was designed to separate content from design.
  • CSS selectors allow you to target HTML elements in various ways.

Git

  • GitHub is a web-based platform that uses Git, a distributed version control system created by Linus Torvalds.
  • GitHub was launched in April 2008 by Tom Preston-Werner, Chris Wanstrath, PJ Hyett, and Scott Chacon.

JavaScript

  • JavaScript is particularly known for its event-driven architecture, which is central to its role in web development.
  • Unlike many other object-oriented languages that use class-based inheritance, JavaScript uses prototype-based inheritance.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
